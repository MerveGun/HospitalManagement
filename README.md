# HospitalManagement# Hospital Management System

## Projeler

1. HospitalManagement.Core
   - Veritabanı modelleri
   - DbContext
   - Ortak servisler

2. HospitalManagement.Admin
   - Admin paneli
   - Doktor yönetimi
   - Branş yönetimi
   - Blog yönetimi
   - Randevu yönetimi

3. HospitalManagement.Web
   - Hastane web sitesi
   - Randevu sistemi
   - Blog
   - İletişim

## Kurulum

1. Repository'yi clone'layın
2. NuGet paketlerini restore edin
3. Migration'ları çalıştırın
4. Projeyi başlatın

## Teknolojiler

- ASP.NET MVC
- Entity Framework
- Bootstrap
- jQuery
